# Cursor拡張機能の推奨設定

このプロジェクトを開発する際に、Cursor（またはVS Code）で推奨される拡張機能を紹介します。これらの拡張機能をインストールすると、コードの編集やデバッグがより快適になります。

## 📦 推奨拡張機能

### TypeScript関連

#### Cursor Pyright（Cursorを使用している場合）

- **説明**: Cursor専用のTypeScript/JavaScript言語サーバーです
- **なぜ必要？**: Cursorを使っている場合、型チェックや自動補完を提供します
- **注意**: 画像に表示されている拡張機能です。Cursorを使っている場合は既にインストールされている可能性があります

### Prisma関連

#### Prisma

- **説明**: Prismaのシンタックスハイライトと自動補完を提供します
- **なぜ必要？**: `prisma/schema.prisma`ファイルを編集する際に、構文エラーをすぐに確認できます
- **注意**: 画像に表示されている拡張機能です。既にインストールされている場合はそのまま使用できます

### Prettier - Code formatter

- **説明**: コードを自動で整形します
- **なぜ必要？**: コードの見た目を統一し、読みやすくします
- **使い方**: `Ctrl+S`（Windows/Linux）または `Cmd+S`（Mac）でファイルを保存すると、自動でコードが整形されます
- **注意**: 画像に表示されている拡張機能です。既にインストールされている場合はそのまま使用できます

### Auto Close Tag

- **説明**: HTML/JSXタグを入力すると、自動で終了タグを追加します
- **なぜ必要？**: `<div>`と入力すると、自動で`</div>`が追加されます。React/JSXの開発で非常に便利です
- **注意**: 画像に表示されている拡張機能です

### Japanese Language Pack for Visual Studio Code

- **説明**: CursorのUIを日本語化します
- **なぜ必要？**: エラーメッセージやメニューが日本語で表示され、初心者にも分かりやすくなります
- **注意**: 画像に表示されている拡張機能です。日本語で開発したい場合に便利です

## 🚀 インストール方法

1. Cursorを開く
2. 左側のサイドバーから「拡張機能」アイコンをクリック
3. 検索バーに拡張機能の名前を入力して検索します
   - 例：`Prisma`と入力すると「Prisma」という拡張機能が表示されます
   - 例：`Prettier`と入力すると「Prettier - Code formatter」が表示されます
   - 例：`Auto Close Tag`と入力すると「Auto Close Tag」が表示されます
4. 目的の拡張機能を見つけたら「インストール」ボタンをクリック

## ⚙️ 推奨設定

拡張機能をインストールしたら、以下の設定を推奨します：

### Cursorの設定を開く

`Ctrl+,`（Windows/Linux）または `Cmd+,`（Mac）で設定を開きます

### 保存時の自動フォーマット

設定画面で以下を有効にします：

1. 検索バーに「format on save」と入力
2. **Editor: Format On Save** にチェックを入れる
3. 検索バーに「default formatter」と入力
4. **Editor: Default Formatter** を `Prettier` に設定

これで、`Ctrl+S`（Windows/Linux）または `Cmd+S`（Mac）でファイルを保存するたびに、自動でコードが整形されます。

## 📝 拡張機能の確認方法

インストールした拡張機能を確認するには：

1. 左側のサイドバーから「拡張機能」を開く
2. 「インストール済み」タブを選択
3. インストールされている拡張機能の一覧が表示されます

## 🆘 トラブルシューティング

### 拡張機能が動作しない場合

1. Cursorを再起動してみてください
2. 拡張機能を一度無効化して、再度有効化してみてください
3. 拡張機能をアンインストールして、再度インストールしてみてください

### TypeScriptの型エラーが表示されない場合

1. `Ctrl+Shift+P`（Windows/Linux）または `Cmd+Shift+P`（Mac）でコマンドパレットを開く
2. `TypeScript: Restart TS Server`
